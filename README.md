# Python_Practice_Questions

A collection of Python practice problems organized by difficulty level and topic.

## Notebooks Overview

### 📘 Easy.ipynb
Beginner-level problems including:
- Palindrome checking
- Armstrong number validation
- Basic number manipulations
- Simple loops and conditionals

### 📗 Medium.ipynb
Intermediate-level problems for developing core skills and problem-solving techniques.

### 📕 Hard.ipynb
Advanced-level problems for mastering complex algorithms and edge cases.

### ⭐ Pattern.ipynb
Pattern-based problems including:
- Star patterns
- Number patterns
- Geometric shapes

### 🔍 Search.ipynb
Search algorithms and techniques:
- Binary search
- Linear search
- Search optimization

### 📊 Sorting.ipynb
Sorting algorithms including:
- Insertion sort
- Selection sort
- Bubble sort
- Other advanced sorting techniques

## Getting Started
Open any notebook and run the cells to practice different Python concepts. Each notebook is self-contained and can be used independently.
